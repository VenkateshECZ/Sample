Code 39 Barcode Symbology Overview

Code 39, A.K.A. Code 3 of 9, ANSI/AIM Code 39, LOGMARS, Alpha39, USD-3, USS-39, Alpha39,

Code 39 is a variable length and self-checking barcode symbology. It is the first alpha-numeric symbology to be developed, and has wildely applications in non-retail industries especially. Code 39 is also the standard barcode used by the United States Department of Defense and the Health Industry Bar Code Council (HIBCC).

Standards: ISO/IEC 16388, ANSI/AIM BC1-1995 USS, BS EN 800, MIL STD 1189, and Code 39 Extended (Code 39 Full ASCII)


 

Install .NET Barcode Reader

  1. Create a project in your Visual Studio.
  2. Unzip the trial package and add "KeepDynamic.BarcodeReader.dll" to project reference.

 

How to read barcodes in .NET applications?

The following C# code illustrates how to read a barcode in local file using .NET class:

using KeepDynamic.BarcodeReader;

string[] barcodeValues = BarcodeReader.read("c://kd-code39.gif", KeepDynamic.BarcodeReader.Type.CODE39);
