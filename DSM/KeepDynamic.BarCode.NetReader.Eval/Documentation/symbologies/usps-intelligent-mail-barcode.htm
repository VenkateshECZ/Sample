Intelligent Mail Barcode Symbology Overview

Intelligent Mail barcode

A.K.A. USPS OneCode, USPS IMb barcode, USPS Intelligent Mail Container Barcode, IM Container Barcode

USPS Intelligent Mail barcode, which was invented to provide greater information and functionality than its predecessors, is used by the United States Postal Service for domestic mail delivery

Standards: USPS-B-3200F

 

Install .NET Barcode Reader

  1. Create a project in your Visual Studio.
  2. Unzip the trial package and add "KeepDynamic.BarcodeReader.dll" to project reference.

 

How to read Intelligent Mail barcodes in .NET applications?

The following C# code illustrates how to read a barcode in local file using .NET class:

using KeepDynamic.BarcodeReader;

string[] barcodeValues = BarcodeReader.read("c://kd-onecode.gif", KeepDynamic.BarcodeReader.Type.ONECODE);
