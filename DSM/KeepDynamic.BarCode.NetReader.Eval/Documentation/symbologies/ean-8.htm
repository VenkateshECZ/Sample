EAN-8 Barcode Symbology Overview

EAN-8, A.K.A. European Article Number 8, GTIN-8, GS1-8, EAN/UCC-8, UCC-8, JAN-8.

EAN-8 is shorter than the European Article Number 13 code. This barcode symbol is primarily used for labling small packages and identify own-brand products sold only in certain stores by great many companies worldwild.

Standards: ISO/IEC 15420, BS EN 797, GS1 General Specifications.


 

Install .NET Barcode Reader

  1. Create a project in your Visual Studio.
  2. Unzip the trial package and add "KeepDynamic.BarcodeReader.dll" to project reference.

 

How to read EAN-8 barcodes in .NET applications?

The following C# code illustrates how to read an EAN-8 barcode in local file using .NET class:

using KeepDynamic.BarcodeReader;

string[] barcodeValues = BarcodeReader.read("c://kd-ean8.gif", KeepDynamic.BarcodeReader.Type.EAN8);
