QR Code Barcode Symbology Overview

QR Code, A.K.A. Quick Response Code, Denso Barcode, QRCode

QR Code is a type of matrix barcode, or two-dimensional code, that has fast readability and large storage capacity than common linear barcode symbology.

Standards: ISO/IEC 18004


 

Install .NET Barcode Reader

  1. Create a project in your Visual Studio.
  2. Unzip the trial package and add "KeepDynamic.BarcodeReader.dll" to project reference.

 

How to read QR Code barcodes in .NET applications?

The following C# code illustrates how to read a QR Code barcode in local file using .NET class:

using KeepDynamic.BarcodeReader;

string[] barcodeValues = BarcodeReader.read("c://kd-qrcode.gif", KeepDynamic.BarcodeReader.Type.QRCODE);
