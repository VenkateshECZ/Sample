UPC-A Barcode Symbology Overview

UPC-A, A.K.A. Universal Product Code version A, GTIN-12, GS1-12, UCC-12

UPC-A, usually carring a GTIN-12, is a linear barcode symbology used for identification of retail goods at point of sale in North America, and in countries like the UK, Australia, and New Zealand.

Standards: ISO/IEC 15420, GS1 General Specifications


 

Install .NET Barcode Reader

  1. Create a project in your Visual Studio.
  2. Unzip the trial package and add "KeepDynamic.BarcodeReader.dll" to project reference.

 

How to read UPC-A barcodes in .NET applications?

The following C# code illustrates how to read a UPC-A barcode in local file using .NET class:

using KeepDynamic.BarcodeReader;

string[] barcodeValues = BarcodeReader.read("c://kd-upca.gif", KeepDynamic.BarcodeReader.Type.UPCA);
