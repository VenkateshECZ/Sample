Codabar Barcode Symbology Overview

Codabar is also known as: Rationalized Codabar, Codeabar, Ames Code, Monarch, 2 of 7 code, ANSI/AIM BC3-1995, NW-7, or USD-4.

Codabar is a linear, discrete, self-checking barcode symbology developed in 1972 by Pitney Bowes Corp. The Codabar symbol is primarily used by libraries and blood banks, photo labs and FedEx airbills and can be accurately read even when printed on dot-matrix printers.

Standards based on the Codabar barcode symbology: Rationalized Codabar, USD-4, NW-7, 2 of 7 Code, ANSI/AIM BC3-1995 USS


 

Install .NET Barcode Reader

  1. Create a project in your Visual Studio.
  2. Unzip the trial package and add "KeepDynamic.BarcodeReader.dll" to project reference.

 

How to read barcodes in .NET applications?

The following C# code illustrates how to read a barcode in local file using .NET class:

using KeepDynamic.BarcodeReader;

string[] barcodeValues = BarcodeReader.read("c://kd-codabar.gif", KeepDynamic.BarcodeReader.Type.CODABAR);
