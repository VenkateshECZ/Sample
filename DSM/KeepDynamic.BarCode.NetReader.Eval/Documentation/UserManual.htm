.NET Barcode Reader Overview

.NET Barcode Reader is an easy to use barcodes recognition component for .NET projects. By using this barcode reader, you are able to add the advanced barcode decoding features into any .NET framework applications.

  • Library/KeepDynamic.BarcodeReader.dll: .NET Barcode Reader component library
  • Documentation/UserManual: Detailed developer guide to scan barcodes in .NET, C# and VB.NET
  • Documentation/symbologies/: .NET barcode examples for reading specific barcode symbologies in .NET
  • /LicenseAgreement: KeepDynamic .NET barcode reader End User License Agreement

    •  

    Supported Barcode Type Symbologies

  • Supported linear barcode standards: compatible with Code 39, Code 39 Extension, Code 128, UPC/EAN (EAN-8, EAN-13, UPC-A, UPC-E), GS1-128 (UCC/EAN-128), Interleaved 2of5, etc.

  • Supported 2D barcode standards: QR Code, Data Matrix, and PDF417

    • .NET Barcode Reader Compatibility

    1. Compatible with all major Windows OS that supports .NET 2.0, 3.0, 3.5 or later version.
    2. Support Microsoft Visual Studio 2005/2008/2010
    3. Support Visual C#, VB.NET programming

     

    Install .NET Barcode Reader

    1. Create a project in your Visual Studio.
    2. Unzip the trial package and add "KeepDynamic.BarcodeReader.dll" to project reference.

     

    How to read barcodes in .NET applications?

    The following C# code illustrates how to read a barcode in local file using .NET class:

    using KeepDynamic.BarcodeReader;

    string[] barcodeValues = BarcodeReader.read("c://kd-code128.gif", KeepDynamic.BarcodeReader.Type.CODE128);
